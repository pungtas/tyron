```own your data, empower your world```

Tyron is a Self-Sovereign Identity (SSI) dapp developed with React Native. Open-source code to build the front-end application for the web browser, iOS and Android.

Main dependencies:
- [Tyron SSI Library](https://github.com/pungtas/tyronzil-js) which integrates [Zilliqa's SDK](https://github.com/Zilliqa/Zilliqa-JavaScript-Library)
- [Expo](https://duckduckgo.com/?q=expo%20react%20native+site:expo.io&t=canonical), [React Native](https://reactnative.dev/) & [React Navigation](https://reactnavigation.org/)

Find the demo on [YouTube](https://www.youtube.com/watch?v=9aiKGehR_gU).
